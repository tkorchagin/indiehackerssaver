# Фоновые изображения для мокапов

Поместите сюда изображения для фона мокапов.

## Как использовать:

1. Добавьте изображение в эту папку (например, `gradient1.png`)
2. Откройте `mockup_config.py`
3. Измените `BACKGROUND_IMAGE = "backgrounds/gradient1.png"`

## Рекомендации:

- Размер: минимум 1920x1080px для лучшего качества
- Формат: PNG или JPG
- Желательно использовать градиенты или абстрактные фоны
- Избегайте слишком ярких или контрастных изображений

## Примеры градиентов:

Вы можете создать градиенты на:
- https://cssgradient.io/
- https://uigradients.com/
- https://coolors.co/gradients

Или использовать готовые фоны с Unsplash, Pexels и т.д.
